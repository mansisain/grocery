# Grocery 

 An ecommerce app to polish react native skills


## Features 


- Authentication
- Categories ->Products ->Product Details
- Database Server(Firestore) + Local(Redux)
- Add/Remove from Cart
- Pull to refresh
- Redux thunk middleware for API
- Remote config
